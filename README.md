# Valorant Agent Randomizer

A simple tool that randomly selects VALORANT agents for fun challenges, custom games, or team compositions.

## Features

* Random agent selection
* Fast and lightweight
* Works entirely in the browser
* No account or installation required

## Tech Stack

* HTML
* CSS
* JavaScript

## Running Locally

1. Clone the repository
2. Open `index.html` in your browser

Alternatively, deploy using GitHub Pages, Vercel, or Netlify.

## Project Structure

```text
index.html
scripts/
styles/
```
